---


### /roleprefix

**Description:** Menambahkan atau menghapus prefix dari nickname member.

### Usage:
- `/roleprefix add`
- `/roleprefix remove`

### Options:
### For `add`:
Menambahkan prefix role tertinggi ke nickname member.
*No options*

### For `remove`:
Menghapus prefix dari nickname member.
*No options*


---


### /sticky

**Description:** atur sticky message

### Usage:

### Options:
### For `set`:
set pesan sticky
